## 기능

- Color picker 색상 선택
- `Hex Copy` 및 `RGBA Copy` 버튼으로 각각 색상 코드를 클립보드에 복사


<br>
<br>

## 개선이 필요한 내용

- 현재 `Hex`부분만 사용자가 직접 Key-in 할 수 있게 되어 있는데 `RGBA`도 필요
- 코드를 좀 더 모듈화
- Typescript 적용
